## Overview

Oftentimes, you might have old and merged Git branches in your local repositories that are needed to clean up. Here is a one-line to remove all of them at once:

```shell
$ git branch --merged | grep -v '\*\|main\|develop' | xargs -n 1 git branch -d
```
