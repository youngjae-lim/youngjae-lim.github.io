## Overview

In this article, we will show a usecase of `hooks` in SvelteKit. In a simple app we are buillding, a user will be able to login and we will authenticate the user and set a cookie. Then whenever the user visits a page, we will check if the cookie is in the request to determine whether or not he/she is authorized to visit the page.
