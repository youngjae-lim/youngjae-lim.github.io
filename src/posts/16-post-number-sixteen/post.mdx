## Overview

Svelte kit and the new Tailwindcss just-in-time compiler just got released. Here's a super quick rundown on how to set them up.

## Steps
1. Set up the project

  ```shell
  $ npm init svelte@next sveltekit-demo
  $ cd sveltekit-demo
  $ npm install -D tailwindcss autoprefixer
  ```

2. Configure postcss

- Create a `postcss.config.cjs` file under the project root directory:

  ```js
  // postcss.config.cjs
  module.exports = {
    plugins: {
      autoprefixer: {},
      tailwindcss: {}
    }
  }
  ```
  
3. Configure tailwind
  - Create a `tailwind.config.cjs` file to enable the just-in-time compiler and add the correct paths to be [purged](https://tailwindcss.com/docs/optimizing-for-production). You can also run `npx tailwind init`

  ```shell
  $ npx tailwind init
  ```

  ```js
  // tailwind.config.cjs
  module.exports = {
    mode: 'jit',
    purge: ['./src/**/*.svelte']
  }
  ```

4. Create and import the css file
  - Add these tags in the `__layout.svelte` so that postcss knows where to insert the tailwind styles.

  ```svelte
  // src/routes/__layout.svelte 
  <script>
	  import Nav from '../components/nav.svelte';
  </script>

  <div class="p-8 max-w-6xl mx-auto">
    <Nav />
    <slot />
  </div>

  <style>
    @tailwind base;
    @tailwind components;
    @tailwind utilities;
  </style>
  ```
